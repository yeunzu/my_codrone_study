### 현재 수정중인 코드
: myCodroneLibrary.py

### 이외 파일 설명
- csv_files : csv 파일 담아두는 디렉터리
- drawing_graph.py : csv파일로 그래프를 통해 경로 그리는 코드.
- drawing_graph_3D.py : csv파일로 그래프를 통해 경로 그리는 코드. 이 코드는 경로를 3D로 보여줌.
- requirements.txt : 필요한 라이브러리를 담은 파일, "pip install -r requirements.txt"를 통해 필요한 라이브러리를 한 번에 받을 수 있음
- temp_trigonmetric_func.py : 드론의 경로 계산에 사용한 삼각함수 활용 관련 코드를 실제 코드에 적용 전 간단한 기능 테스트를 한 코드.
